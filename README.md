# 💜✨ Análise de Dados: Gênero, Etnia, Escolaridade e Salários no Brasil

Projeto desenvolvido no bootcamp de Análise de Dados da PrograMaria, com o objetivo de explorar dados e gerar reflexões sobre desigualdades sociais no Brasil através da análise de dados 📊💭

---

## 🌙🔍 Objetivo

Investigar, de forma exploratória, possíveis diferenças relacionadas a:

Gênero, Etnia/Raça, Escolaridade, Salário (e outros dados)

Buscando entender como essas variáveis se relacionam e quais padrões podem ser observados nos dados.

---

## 🧰🪄 Ferramentas utilizadas

Python
Google Colab
DBeaver (SQL e manipulação de banco de dados)
Looker Studio (visualização e dashboards)
Pandas (tratamento e análise de dados)
Matplotlib (visualizações)
Seaborn (visualizações estatísticas)

---

## 📂🫧 Base de dados

A base utilizada contém informações sobre pessoas no Brasil, incluindo:

* Gênero
* Etnia/Raça
* Idade
* Escolaridade
* Salário
* Estado

---

## 📈🌸 Análises realizadas

* Distribuição de pessoas por gênero e etnia
* Comparação de salários entre diferentes grupos
* Relação entre nível de escolaridade e renda
* Cruzamento de variáveis (ex: gênero x salário, etnia x escolaridade)
* Visualizações para facilitar a interpretação dos dados

---

## 📊🎨 Visualizações

O projeto inclui:

Gráficos de barras para comparação entre grupos
Gráficos de dispersão (ex: idade x salário)
Mapas de distribuição geográfica
Heatmaps para análise de correlação

---

## 💡✨ Principais insights

* Observa-se diferenças salariais entre gêneros, sugerindo possíveis desigualdades estruturais
* Há variações relevantes de renda entre diferentes grupos étnicos
* O nível de escolaridade apresenta relação com o aumento da renda
* Alguns grupos aparecem com menor presença em níveis mais altos de escolaridade
* Os dados indicam padrões que refletem aspectos sociais e estruturais da sociedade brasileira

---
